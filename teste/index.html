Aprendendo a manipular imagens

As imagens podem assumir vários formatos, cada um melhor para uma aplicação, por exemplo:

SVG - formao usado para vetores, como logos ou imagens simples

JPEG - formato com a maior qualidade, mas mais pesado

PNG - inferior ao JPEG, mas pode ser comprimido e manter a qualidade, além disso tem a vantagem do fundo transparente

WEBP - formato criado especificamente para web pelo google

Fonte: Wikipédia

Atributos de imagens

o tamanho da imagem também pode ser alterado através dos atributos height e width.

Outro atributo importante ao se trabalhar com imagens é o alt, ele funciona dando um contexto para o usuário, para caso a imagem não apreça, ou permitir acessibilidade para usuários que tem algum tipo de dificiência visual.
